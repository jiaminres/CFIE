# CFIE 文档导航

## 1. 文档结构

当前 `docs/` 只保留三类长期有效文档：

- `路线文档/`
  - 说明训练侧与推理侧四条主线的目标、范围、现状与代码入口。
- `架构图文档/`
  - 说明关键运行链路、架构图、Windows 适配计划与执行台账。
- `历史归档/`
  - 保留少量阶段总结与验证基线，避免零散交接记录长期堆在顶层。

- 顶层不再保留零散的会话交接文档。
